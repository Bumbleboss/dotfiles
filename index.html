About

This is my collection of configuration and personalization files for various applications I use on my system.

There's a very high probability that everything contained in this repo was blatantly stolen from one of the following places:

  • Ryan Bates' dotfiles
  • Zach Holman's dotfiles
  • Ryan Tomayko's dotfiles
  • thoughtbot's dotfiles

Installation

$ git clone https://github.com/cobyism/dotfiles ~/.dotfiles
$ cd ~/.dotfiles
$ rake

Use at your own peril.
